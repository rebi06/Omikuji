
# Omikuji - Java おみくじアプリ 🎴

このプログラムは **条件分岐と繰り返し（if / switch / while）** の練習として作成した  
シンプルなおみくじアプリです。  
学習目的：Java 第3章「条件分岐と繰り返し」の理解と実践。

---

## 🎯 プログラム概要

ユーザーが「s」を入力するとおみくじを引き、  
「q」を入力すると終了します。  
毎回ランダムに「大吉」「中吉」「小吉」「吉」「凶」のいずれかが表示されます。

## 学んだこと
① Random の使い方

当初：毎回 new Random() を作っていた。
問題点：毎回インスタンスが作られて無駄が多い。
改善：クラス内で private static final Random RANDOM = new Random(); にして使い回し。
→「オブジェクトを1つだけ使う」という考え方を身につけた。

② JavaDoc の書き方

当初：引数のないメソッドに @param が書かれていた。
問題点：JavaDocのタグの意味を理解しきれていなかった。
改善：@return だけに整理。コメントも短く「何をするメソッドか」を書くようにした。
→ JavaDocの正しい使い方を習得。

③ Scanner のリソース解放

当初：閉じていなかった or 忘れがち。
改善：sc.close(); を明示的に追加。
→ Javaのリソース管理の基本を理解。

④ ラベル付き break の使い方

当初：outer: は使えていたが、意味を曖昧にしていた。
改善：ラベルの目的（「外側ループだけ抜ける」）を明確に理解。
→ ネスト構造の中で「どのループを抜けるか」を正しくコントロールできるようになった。
